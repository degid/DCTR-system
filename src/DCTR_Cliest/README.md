# DCTR-system


## Installation

You can either download the source and compile or use nuget at http://nuget.org. To install with nuget:

	Install-Packages:
	Google.Apis.Drive.v3
	{
		BouncyCastle
		log4net
		Microsoft.Bcl.Build
		Microsoft.Bcl
		Microsoft.Bcl.Async
		Microsoft.Net.Http
		Newtonsoft.Json
		Google.Apis.Core
		Google.Apis.Auth
		Zlib.Portable.Signed
		Google.Apis
	}
	TaskScheduler
